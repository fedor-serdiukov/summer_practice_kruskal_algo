# Прототип (0-я версия)

## Что реализовано
- Главное окно с тремя областями: панель управления
  (сверху), рабочая область графа (в центре), панель информации (справа),
  журнал работы (снизу).
- Панель управления: кнопки «Создать граф», «Загрузить»,
  «Сохранить», «Очистить»; выпадающий список режимов; кнопки «Запустить алгоритм»,
  «Предыдущий шаг», «Следующий шаг», «Автовыполнение», «Сброс»; регулятор
  скорости (задержка, мс).
- Отображение графа: вершины кругами с номерами, рёбра линиями с подписями
  весов (демонстрационный граф из 5 вершин и 7 рёбер).

## Сборка и запуск
    javac -encoding UTF-8 -d out src/kruskal/*.java
    jar cfm KruskalVisualizer-v0-prototype.jar MANIFEST.MF -C out .
    java -jar KruskalVisualizer-v0-prototype.jar
